ExoCTK (the Exoplanet Characterixation Tool Kit) is an open-source, modular, data analysis software package focused primarily on atompspheric characterization of exoplanets and observation planning. This web application runs the most recent available version of ExoCTK through this web portal designed to single out the most useful tools for observation planning, forward modeling, data reduction, limb darkening, light curve fitting, and retrievals.
